  • HTML Made Simple: Dive into Web Development
  • Yousef Khaled Al Hinai
  • 5-8 / 07 / 2024
  • 09:30 - 10:30 pm
×

Descrpiton:

Join us for "HTML Made Simple: Dive into Web Development," where you'll embark on an exciting journey into the world of HTML (Hypertext Markup Language). This workshop is designed for beginners eager to learn the fundamental building blocks of web pages. Whether you're aiming to build your first website or enhance your existing skills, this session provides a solid foundation in HTML essentials.

Objectives:

- Introduction to HTML: Learn the syntax and structure of HTML for web development.
- Paragraphs: Format text using HTML paragraph tags.
- Headings: Organize content with HTML heading tags.
- Images and Figures: Add images and figures to web pages.
- Links: Create hyperlinks for navigation.
- Navigation (nav) and Sidebars (aside): Implement navigation bars and sidebars.
- Citations (cite) and Quotes: Properly format citations and quotes.
- Lists: Organize information with unordered and ordered lists.
- Address and Time: Use HTML tags for addresses and time-related content.
- Divisions (divs): Structure content with div tags.
- Tables: Present data using HTML tables.
- Forms: Build interactive forms for user input.
- Title, Comments, and Meta Tags: Manage web page titles, developer notes, and meta tags.
- Project Integration: Apply HTML concepts in a practical project.
